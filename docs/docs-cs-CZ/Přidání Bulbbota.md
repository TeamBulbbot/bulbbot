# Přidavaní Bulbbota
Přidat BulbBota můžete lehce přes tento [odkaz](https://bulbbot.mrphilip.xyz/invite)

### Permission Breakdown
Bulbbot potřebuje ze základu následující oprávnění: 

- Správa rolí
- Správa kanálů
- Vyhodit člena
- Zabanovat členy
- Vytvořit pozvánku
- Spravovat Přezdívky
- Správa Přezdívek
- Správa smajlíků
- Zobrazit protokol auditu
- Read Messages
- Vkládat soubory
- Číst historii zpráv
- Přidat reakce
- Používat externí smajlíky
- Vypnout členům mikrofon
- Vypnout členům zvuk
- Přesunout uživatele
