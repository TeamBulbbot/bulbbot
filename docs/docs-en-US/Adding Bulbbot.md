# Adding Bulbbot
Adding Bulbbot to your server is easily done with this [direct invite link found here](https://bulbbot.mrphilip.xyz/invite)

### Permission Breakdown
Bulbbot will require these permissions by default

- Manage Roles
- Manage Channels
- Kick Members
- Ban Members
- Create Invite
- Manage Nickname
- Change Nickname
- Manage Emojis
- View Audit Log
- Read Messages
- Attach Files
- Read Message History
- Add Reactions
- Use External Emojis
- Mute Members
- Deafen Members
- Move Members
